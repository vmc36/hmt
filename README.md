# Help My Trip! (React + Vite + TypeScript)

App para estimar custo de viagem para Londres, converter USD para GBP (libra esterlina) com cotação atual e calcular quanto poupar por mês até novembro.
Passagem e hospedagem podem ser informadas como valores totais em BRL.

## Stack

- React 18
- Vite 5
- TypeScript 5
- CSS moderno e responsivo
- API de câmbio: Frankfurter (`GBP -> USD, BRL`)

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
